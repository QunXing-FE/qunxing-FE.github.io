群星开发中爬过的那些坑

发表于 2016-07-11   |   分类于 总结   |  

还记得曾经因为一个小小的问题百撕不得其姐吗？最后浪费我们几个小时甚至一下午，更甚一天或者更久的时间，下面我们就总结下在开发中爬过的那些坑吧(说多了都是泪啊)。

bootstrap模态框和select2合用时input无法获取焦点

描述：在bootstrap的模态框里使用select2插件,会导致select2里的input输入框没有办法获得焦点,没有办法输入.
解决方案：把页面中的 tabindex=”-1” 删掉
重写enforceFocus方法:$.fn.modal.Constructor.prototype.enforceFocus = function() {};

收银台启动程序无法启动

描述：找到项目的startup程序，右键run as 没有出现java application
原因分析：项目导入时是从EGit作为git项目导入进来
解决方案：把项目作为maven项目重新导入解决

点击一个元素后，按空格键会一直触发该元素的绑定事件

描述：点击一个元素后，按空格键会一直触发该元素的绑定事件
原因分析：调查中
解决方案：点击空格键return出去

1001登录加载静态文件失败

描述：1001登录企业端 handlebars加载的静态资源 200/206
原因分析：eclipse Text file encoding没有设置为UTF-8
解决方案：encoding设置为UTF-8
